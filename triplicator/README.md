# Triplicator

## Description 
 A collection of objects and functions that gives control over naming and formatting of variables during bibliography to triple (i.e., RDF) conversion.
    
## Requirements
- Pyhon 3.x
- pybtex

## Currently supported input formats
- .bib
- .csv
